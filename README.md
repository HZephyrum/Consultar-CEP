# Consultar CEP

Essa ferramenta foi criada no intuito de agilizar o processo de consulta de CEP's e Faixas Cep sem precisar ter que acessar o site dos correios pra isso.

### Tecnologias Utilizadas:
* Linguagem: C#  
* WPF
* Material Design for XAML
* Refit
* Newtonsoft.Json
